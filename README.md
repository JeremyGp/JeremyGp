## Featured Projects

### StudyGo

An intelligent academic planning system designed to help students organize and manage their academic workload more effectively.

StudyGo helps students break down complex assignments into manageable subtasks, recommends when they should start working based on upcoming deadlines, and provides personalized reminders to reduce delays and improve time management.

**Backend:** Python · FastAPI · SQLAlchemy · MySQL

**Key features:**

* Academic task and deadline management
* Automatic generation of subtasks for complex assignments
* Recommended start dates based on task deadlines
* Personalized reminders and alerts
* User authentication and authorization
* REST API built with FastAPI
* Relational database integration

🔗 [Backend Repository](https://github.com/JeremyGp/StudyGo_b)

### MindU

A university-focused platform designed to facilitate access to psychological support by allowing students to schedule and manage appointments with university psychologists.

**Backend:** Java · Spring Boot · PostgreSQL

Key features:

* Student and psychologist management
* Appointment scheduling
* Virtual and in-person appointment modalities
* Appointment status management
* Relational database integration

🔗 [Repository](https://github.com/JeremyGp/mindu)
